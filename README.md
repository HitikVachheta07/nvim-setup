# Neovim Shortcuts 

##  File Explorer (NvimTree)
| Key             | Action                 |
|-----------------|------------------------|
| `Enter`         | Open file              |
|  a              | New file               |
|  d              | Delete                 |
|  r or e         | Rename                 |
|  shift + h      | Toggle hidden files    |
| :NvimTreeToggle | Toggle file tree       |
|  Ctrl + b       | Toggle file tree       |
|  f              | find file              |
|  Ctrl + b       | Toggle file tree       |
|  Ctrl + s       | Save file              |
|  \ + w          | Close file              |

---

##  Window Management
| Key              | Action           |
|------------------|------------------|
| :split  or \ + s | Horizontal split |
| :vsplit or \ + v | Vertical split   |

## Move Between splits
| ctrl + h  | previous split |
| ctrl + l  | next split     |

---

##  Navigation
| Key       | Action       |
|-----------|--------------|
| Alt + l   | Next tab     |
| Alt + h   | Previous tab |

---

## Terminal
| Key                    | Action                  |
|------------------------|-------------------------|
| Ctrl + \               | Toggle terminal         |
| Ctrl + t               | Toggle terminal         |
| \ + 1                  | Switch Toggle terminal  |
| \ + 2                  | Switch Toggle terminal  |
| \ + 3                  | Switch Toggle terminal  |
| Ctrl + Shift + V       | Paste                   |
| Ctrl + Shift + C       | Copy                    |
| " :split | terminal "  | Horizontal split        |
| " :vsplit | terminal " | Vertical split          |

---

## Editing
| Key           | Action      |
|---------------|-------------|
| yy            | Copy (yank) |
| p             | Paste       |
| u             | Undo        |
| Ctrl + r      | Redo        |
| :%s/old/new/g | Replace all |

---

## Search
| Key      | Action        |
|----------|---------------|
| Ctrl + f | Global search |
| Ctrl + p | File search   |

---

##  Folding
| Key| Action      |
|----|-------------|
| zc | Collapse    |
| zo | Expand      |
| za | Toggle fold |

---

## Save & Exit
| Key   | Action       |
|-------|--------------|
| q     | Exit         |
| :wq   | Save & exit  |
| :qa!  | Force exit   |
