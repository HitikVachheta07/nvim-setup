# nvim-setup
Full Nvim setup

# Neovim Shortcuts 

##  File Explorer (NvimTree)
| Key             | Action                 |
|-----------------|------------------------|
| `Enter`         | Open file              |
|  a              | New file               |
|  d              | Delete                 |
|  r              | Rename                 |
|  H              | Toggle hidden files    |
| :NvimTreeToggle | Toggle file tree       |
|  Ctrl + b       | Toggle file tree       |

---

##  Window Management
| Key     | Action           |
|---------|------------------|
| :split  | Horizontal split |
| :vsplit | Vertical split   |

---

##  Navigation
| Key       | Action   |
|-----------|----------|
| Shift + L | Next tab |

---

## Terminal
| Key                    | Action           |
|------------------------|------------------|
| Ctrl + \               | Toggle terminal  |
| Ctrl + Shift + V       | Paste            |
| Ctrl + Shift + C       | Copy             |
| " :split | terminal "  | Horizontal split |
| " :vsplit | terminal " | Vertical split   |

---

## Editing
| Key           | Action      |
|---------------|-------------|
| yy            | Copy (yank) |
| p             | Paste       |
| u             | Undo        |
| Ctrl + r      | Redo        |
| :%s/old/new/g | Replace all |

---

## Search
| Key      | Action        |
|----------|---------------|
| Ctrl + f | Global search |
| Ctrl + p | File search   |

---

##  Folding
| Key| Action      |
|----|-------------|
| zc | Collapse    |
| zo | Expand      |
| za | Toggle fold |

---

## Save & Exit
| Key   | Action       |
|-------|--------------|
| q     | Exit         |
| :wq   | Save & exit  |
| :qa!  | Force exit   |
